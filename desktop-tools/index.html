Desktop Tools

NCS Pcoder Add-in for Excel

NCS Pcoder is an add-in for Microsoft Excel which can help you match place names in Excel table and insert NCS codes automatically according to the GAD’s Place Names and Codes.

Python based tools

NCS Pcoder Python tools can help you match place names in various table and spatial formats such as shp, geojson, excel, csv, spatial format that can be opend in ArcGIS and QGIS. These tools are experimental and need python knowledge.
